# 🚀 Space Adventure CLI

A fun and beginner-friendly command-line space exploration game built with Python.

## 🌟 Features

* 🌍 Explore mysterious planets
* 👾 Encounter alien attacks
* 💎 Find hidden treasures
* ✨ Discover energy crystals
* ❤️ Manage your health
* ⛽ Manage your fuel
* 🏆 Earn points and achieve a high score
* 🎮 Simple and interactive CLI gameplay

## 📂 Project Structure

```text
Space-Adventure-CLI/
├── Space-Adventure-CLI.py
└── README.md
```

## ▶️ How to Run

1. Open a terminal.
2. Navigate to the project directory:

```bash
cd games/Space-Adventure-CLI
```

3. Run the game:

```bash
python Space-Adventure-CLI.py
```

## 🎮 Gameplay

* Choose **Explore Planet** to travel to a new planet.
* Each exploration may result in:

  * 👾 Alien attack
  * 💎 Treasure discovery
  * ✨ Energy crystals
  * 🌌 Empty planet
* Use **Scan Space** to discover interesting space events.
* Survive as long as possible while managing your health and fuel.
* Earn points and try to get the highest score.
